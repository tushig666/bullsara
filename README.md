## English

### Project Overview
Bullsara is a premium e-commerce and lottery platform designed for luxury car enthusiasts. Users can participate in high-stakes raffles to win exclusive luxury vehicles. The platform features a seamless user experience, real-time data updates, and a robust administrative backend.

### Key Features
*   **User Authentication:** Secure login and signup using Firebase Authentication.
*   **Real-time Product Listings:** Browse available luxury cars with real-time stock and status updates.
*   **Secure Participation:** Simulated payment flow (QPay integration) for purchasing lottery entries.
*   **Profile Management:** Personalized user profiles to track orders and update information.
*   **Admin Dashboard:** Comprehensive tools for administrators to manage products, monitor orders, and handle user roles.
*   **AI Integration:** Powered by Genkit for intelligent features and automation.

### Tech Stack
*   **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS.
*   **UI Components:** ShadCN UI, Lucide React Icons.
*   **Backend & Database:** Firebase Firestore, Firebase Authentication.
*   **AI Framework:** Genkit with Google AI (Gemini).
*   **Deployment:** Optimized for Firebase App Hosting and Vercel/Netlify.

### Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/tushig666/bullsara-bitch.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up Environment Variables:**
    Create a `.env.local` file and add your Firebase configuration and Gemini API Key:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```

## Монгол

### Төслийн танилцуулга
Bullsara бол тансаг зэрэглэлийн автомашин сонирхогчдод зориулсан онлайн худалдаа болон сугалааны платформ юм. Хэрэглэгчид онцгой, чамин хүлэгүүдийг хожих боломжтой сугалаанд оролцох боломжтой. Платформ нь ашиглахад хялбар интерфэйс, бодит цагийн мэдээлэл шинэчлэлт болон удирдлагын цогц системтэй.

### Үндсэн боломжууд
*   **Хэрэглэгчийн бүртгэл:** Firebase Authentication ашиглан аюулгүй нэвтрэх, бүртгүүлэх.
*   **Бүтээгдэхүүний жагсаалт:** Боломжит машинуудыг бодит цагийн үлдэгдэл, статустай харах.
*   **Сугалаанд оролцох:** Сугалааны тасалбар авах аюулгүй төлбөрийн симуляци (QPay).
*   **Профайл удирдлага:** Захиалгын түүхээ хянах, хувийн мэдээллээ шинэчлэх.
*   **Админ самбар:** Бүтээгдэхүүн нэмэх, захиалга хянах, хэрэглэгчийн эрх удирдах хэрэгслүүд.
*   **AI систем:** Ухаалаг үйлдлүүд болон автоматжуулалтад зориулсан Genkit интеграци.

### Ашигласан технологиуд
*   **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS.
*   **UI Компонентууд:** ShadCN UI, Lucide React Icons.
*   **Backend & Өгөгдлийн сан:** Firebase Firestore, Firebase Authentication.
*   **AI Фрэймворк:** Genkit (Google AI Gemini).
*   **Байршуулалт:** Firebase App Hosting, Vercel эсвэл Netlify-д зориулж оновчлогдсон.
