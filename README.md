# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Adding a Custom Logo

To make your custom logo appear in the header, please follow these steps:

1.  Create a folder named `public` in the root directory of your project.
2.  Place your logo image file inside this `public` folder.
3.  Make sure the image file is named `logo.png`.

The application is already configured to automatically use the image from the `/public/logo.png` path.
